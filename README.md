# Amina Happiness Meter

A warm, playful daily mood tracker built with Expo, React Native, TypeScript, Expo Router, NativeWind, TanStack Query, Zustand, React Hook Form, and a Supabase-ready data layer.

## Run locally

```bash
pnpm install
pnpm dev
```

Press `w` for the web app, or open it with Expo Go / an iOS or Android simulator.

## Optional services

Copy `.env.example` to `.env` and add Supabase credentials to enable a hosted backend. The current experience stores check-ins locally, so the app is useful immediately with no account or API key.

## Product note

This is a wholesome wellbeing journal, not a medical or diagnostic product. Its insights are deliberately supportive and non-clinical.
